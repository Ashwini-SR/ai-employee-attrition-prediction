# AI Employee Attrition Prediction System

Hackathon Project – Novathon

## Overview
This system predicts employee attrition risk using Machine Learning and provides HR recommendations using Generative AI.

## Features
- Attrition prediction model
- HR risk scoring
- AI retention suggestions
- Interactive HR dashboard
